# 群聊拟人模仿复读插件

AstrBot 插件，自动学习群消息并随机模仿群成员发言。

## 功能
- 自动学习群内文本与图片消息
- 按概率随机模仿（支持图片）
- 每群独立冷却，消息权重机制
- 每月自动清理过期数据
- `/ai清理` 私聊指令（白名单）

## 配置项
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 数据目录 | ./data/AI66 | 数据存储路径 |
| 模仿概率 | 0.12 | 触发概率 0~1 |
| 冷却秒 | 20 | 每群冷却时间 |
| 最大保存字数 | 200 | 单条消息保存上限 |
| 允许清理的QQ号 | [] | 可执行清理的QQ号 |
